Ordered list
Used to group a collection of items together in a list. Each list item is defined by a <li> element, which suggests an order of importance or sequence.
Unordered list
The <ut> element, the name for which is an abbreviation of unordered list, is used to group a collection of items together in a list, but in a way that doesn't suggest an order of precedence or importance.
Dictionary list
If you want to list a series of items that essentially have a title and a description of some kind. Encapsulated by <dl> tags, use <dt> for the 'word' and <dd> for the definition.
