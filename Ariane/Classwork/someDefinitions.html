Some Definitions

HTML
HTML stands for Hyper Text Markup Language. It is the primary language used to create web pages. XHTML is an updated version of HTML.
Markup
A set of tags that suggest the structure of a document. For example: What is a heading, paragraph, list, etc.
CSS
CSS stands for Cascading Style Sheets. CSS is a language that lets you control how your web pages look.
Web Standards
Web Standards advocate best practices for building web pages. These are published by the world wide web consortium (W3C).
